# 参考图索引

> 每张参考图都在这里记录描述和喜欢/不喜欢的点。
> Agent 通过这个索引找到相关参考图，然后用 Read 工具查看图片。

## 目录说明

- `liked/` — 喜欢的设计截图
- `disliked/` — 不喜欢的设计（反面教材）
- `palettes/` — 配色方案截图
